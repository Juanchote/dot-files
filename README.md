dot-files
=========

Archivos de configuracion de diferentes programas

__Sublime-Text__
> Ir a 'Preferences', buscar la opción y pegar ahí el contenido.

__vim__
> ~/.vimrc

__git__
> ~/.gitconfig

__tmux__
> ~/.tmux.conf

__bashrc__
> Añadir a ~/.basrc, al final del archivo mismamente

__mocp__
> Copiar en /usr/share/moc/themes/
